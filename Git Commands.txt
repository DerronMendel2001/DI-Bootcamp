Creating a Repository
1. Create the repo in Git
2. In terminal, go to folder you want to push
3. git init
4. git add .
5. git commit -m "push name"
6. git branch -M main
7. git add remote [url]
8. git push origin main

Push to an existing repo
1. git add .
2. git commit -m "push name"
3. git push origin main
